# WAN (Wide Area Network)

> 회선 교환, 패킷 교환, 셀룰러 네트워크, 이동통신 세대

## 스위칭 통신 네트워크

- **스위칭 노드**: 노드 간 데이터를 이동시키는 스위칭 시설
- **스테이션**: 네트워크에 연결된 장치
- **노드**: 통신을 제공하는 스위칭 장치

## 회선 교환 (Circuit Switching)

- 두 스테이션 간 **전용 경로** 사용
- 3단계: 설정(Establish) → 전송(Transfer) → 해제(Disconnect)
- 특징:
  - 연결 지속 시간 동안 채널 용량 전용 → 데이터 없을 때 낭비
  - 아날로그 음성 신호 전송에 효율적
  - 연결 후 직접 연결처럼 투명하게 작동

### 블로킹 vs 비블로킹
- **블로킹 네트워크**: 모든 경로 사용 중 시 연결 불가 (음성 시스템에 사용)
- **비블로킹 네트워크**: 모든 스테이션 동시 연결 가능 (데이터 연결에 필요)

### 스위칭 기술
- **공간 분할 스위칭**: 신호 경로가 물리적으로 분리
- **시간 분할 스위칭**: 디지털 시간 분할로 가상 회선 설정

## 패킷 교환 (Packet Switching)

- 회선 교환은 음성용, 패킷 교환은 **데이터용**으로 설계
- 작은 패킷으로 분할 전송 → 각 패킷에 사용자 데이터 + 제어 정보(라우팅 주소)
- 패킷은 수신 후 버퍼링 → 다음 노드로 전달

### 패킷 교환의 장점
- **회선 효율성**: 단일 링크를 여러 패킷이 시간 공유
- **데이터 속도 변환**: 스테이션은 자체 속도로 로컬 노드에 연결
- 네트워크 혼잡 시에도 패킷 수용
- 우선순위 적용 가능

### 패킷 처리 방식

| 방식        | 설명                | 특징                       |
| --------- | ----------------- | ------------------------ |
| **데이터그램** | 각 패킷이 독립적으로 처리    | 호출 설정 단계 없음, 더 유연, 더 신뢰적 |
| **가상 회선** | 미리 계획된 경로 설정 후 전송 | 패킷이 빠르게 전달, 덜 신뢰적        |

## 셀룰러 네트워크 (Cellular Networks)

### 구성
- 지역을 **셀**로 분할 (타일 패턴)
- 각 셀에 고유 주파수 범위 할당
- 인접 셀은 간섭 방지를 위해 다른 주파수 사용
- 충분히 먼 셀은 동일 주파수 재사용 가능 (Frequency Reuse)

### 채널 유형
- **제어 채널**: 호출 설정 및 유지
- **트래픽 채널**: 음성 및 데이터 전송

### 전파 효과 (Propagation Effects)
- 반사, 산란, 회절 3가지 메커니즘

#### 페이딩 종류
| 종류          | 설명                            |
| ----------- | ----------------------------- |
| **빠른 페이딩**  | 반파장 거리 내에서 신호 강도 급격히 변동       |
| **느린 페이딩**  | 건물·공터 등 환경 변화에 따라 평균 수신 전력 변화 |
| **평탄 페이딩**  | 모든 주파수 성분이 동일 비율로 동시 변동       |
| **선택적 페이딩** | 신호 대역폭 일부에서만 감쇠 발생            |

#### 오류 보상 메커니즘
- **순방향 오류 수정 (FEC)**: 추가 비트로 오류 수정
- **적응형 등화**: 심볼 간 간섭 극복
- **다양성 (Diversity)**: 여러 채널로 분산 전송
  - 주파수 다양성, 시간 다양성, 스프레드 스펙트럼

## 이동통신 세대

| 세대                    | 특징                                       |
| --------------------- | ---------------------------------------- |
| **1G**                | 아날로그, 음성 전용 (AMPS)                       |
| **2G**                | 디지털, 암호화, TDMA/CDMA 방식                   |
| **3G**                | 멀티미디어, 고속 무선 통신, CDMA 지배                 |
| **4G (LTE-Advanced)** | 전체 IP 패킷 교환, 100Mbps~1Gbps, OFDMA + MIMO |

### LTE-Advanced
- **OFDMA**: 직교 주파수 분할 다중 접속
- **MIMO**: Multiple-Input Multiple-Output 안테나
- TDD (시간 분할 이중) / FDD (주파수 분할 이중)
- Femtocells: 저전력, 단거리, 자립형 기지국
