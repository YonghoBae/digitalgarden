## 정의
- 여러 자원을 묶어 마치 하나의 거대한 자원으로 활용할 수 있는 기술
- 자원은 동일한(homogeneous) 자원일 수도 있고 다른 상이한(heterogeneous)자원일 수도 있음
- 일반적으로 클러스터링에서 동일한 IT자원을 네트워크로 연결하는 것을 의미
- 독립적인 IT 자원을 연결하여 마치 하나의 시스템과 같이 동작하도록하는 기술

## 클러스터링
- 개별 컴퓨터를 묶어 하나의 목적으로 활용
- 느슨하게 또는 강력하게 연결되어 작업을 처리하는 컴퓨터
- 단일 시스템처럼 보임
- 그리드 컴퓨터와 다르게, 각 노드는 같은 작업을 처리함

## 구조
- 컴퓨팅 자원은 LAN를 통해 통신
- 컴퓨팅 자원을 통제하는 마스터 노드 존재
- 마스터 노드는 별도 컴퓨터 or 일반 컴퓨티 자원에 마스터 역할 부여
- 스토리지 전용 네트워크 SAN를 통해 통신
