## 정의
- 가상 인스턴스를 만드는 기술
- 컴퓨터 리소스의 추상화를 일컫는 광범위한 용어
- [[02 - Knowledge/Computing/Cloud Computing/기술/하이퍼바이저\|하이퍼바이저]](소프트웨어)를 사용하여 하나의 물리적 머신에서 가상 머신(VM)을 만드는 프로세스
	- VM은 물리적 머신과 동일한 역할과 성능 수행
	- 물리적 머신의 컴퓨팅 리소스를 사용
	- 하이퍼바이저는 필요에 따라 각 VM에 컴퓨팅 리소스 할당
- 서버 가상화를 통해 컴퓨터에서 동시에 1개 이상의 운영체제 가동
	- 대부분의 서버 = 용량의 10~15%만 사용
	- 서버의 효용률 70% up
- 높은 효용률 -> 동일한 처리량을 위한 컴퓨터 수를 줄여줌
