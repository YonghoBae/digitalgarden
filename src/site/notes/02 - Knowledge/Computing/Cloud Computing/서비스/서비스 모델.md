- 서비스 모델은 서비스를 받는 사용자 관점에서 기술
## IaaS (Infrastructure as a Service)
  - 기본 인프라 제공 (서버, 스토리지, 네트워크 등)
  - 인프라스트럭처 = 인프라 = 컴퓨팅 인프라 = 서버, 네트워크, 스토리지 자원 등
  - 물리시스템을 서비스로 제공
  - 일반적으로 물리시스템 직접 제공보다 가상화 기술을 이용하여 컴퓨팅 인프라를 가상 자원으로 제공
  - 예시: AWS EC2, Azure VM

## PaaS (Platform as a Service)
  - 플랫폼을 서비스하는 모델
  - 프로그램 개발자와 같은 특정 그룹에 중점
  - 동일한 개발환경을 플랫폼으로 제공 
  - 예시: Google App Engine

## SaaS (Software as a Service)
  - 소프트웨어를 사용한 만큼 비용 지불
  - 개별 사용자에 중점
  - 소프트웨어 서비스를 다양한 클라이언트 장치 or 웹 인테페이스로 접근
  - 사용자가 직접 설치X
  - 사용자가 업데이트, 관리 필요x
  - 예시: Gmail, Dropbox
