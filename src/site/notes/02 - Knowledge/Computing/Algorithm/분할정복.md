## 설계 전략
- 분할(Divide) : 해결할 문제를 여러 개의 작은 부분으로 나눈다.
- 정복(Conquer) : 나눈 작은 문제를 각각 해결한다
- 통합(Combine) : (필요하다면) 해결된 해답을 모은다.

## 반복(Iterative) 알고리즘: O(N)
```pascal
Iterative_Power(x, n)
	result <- 1
	
	FOR i in 1 -> n
		result <- result * x
	
	RETURN result
```

## 분할 정복 기반의 알고리즘: O(logN)
```pascal
Recursive_Power(x, n)
	IF n  == 1 : RETURN x
	IF n is even
		y <- Recursive_Power(x, n/2);
		RETURN y*y
	ELSE
		y <- Recursive_Power(x, (n-1)/2)
		RETURN y*y*x
```
