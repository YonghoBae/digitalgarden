# Union-Find (서로소 집합 연산)

> Make-Set / Find-Set / Union 연산 구현

## 핵심 연산

- **Make-Set(x)**: 원소 x만 포함하는 집합 생성
- **Find-Set(x)**: x가 속한 집합의 대표자(루트) 반환
- **Union(x, y)**: x와 y가 속한 두 집합을 합침
